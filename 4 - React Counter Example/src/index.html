In the previous Counter Example we hard coded the dom update manualy, and of course this approach does not skill to complex applications,

There for, instead of manually update the DOM we're going to use React

To Start with react, we're going to use Babel to transpile the code and webpack for Module Bundling, you check the 'webpack.config.js' and 'package.json' for better understanding which tools we're using for this example

for now just run 'yarn' to install the dependencies then run 'yarn start' to start the project
